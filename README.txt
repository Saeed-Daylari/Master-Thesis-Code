This repository contains the Python implementation used in my Master’s thesis.

Title: Entwicklung einer reaktiven Paste mit definierten rheologischen Eigenschaften und variablem Auflöseverhalten
Author: Saeed Ghorbani Daylari
University: HTW Dresden
Year: 2026

Description

This repository provides the implementation of data-driven surrogate models based on Artificial Neural Networks (ANNs) for fitting and analyzing experimental rheological data of reactive pastes. The models are used to capture nonlinear relationships between process parameters and material behavior. 

Requirements

Python ≥ 3.3
